# Password Reset Client

Frontend application for the Password Reset project, built using **React** and **Tailwind CSS**.  
This client connects to the backend API for user authentication and password reset functionality.

---

## 🚀 Features

- User Login
- User Registration
- Forgot Password Flow
- Reset Password via Email Token
- Tailwind CSS UI Styling
- Simple Client-Side Routing

---

## 🛠 Tech Stack

- React
- Tailwind CSS
- Axios
- Fetch API
- Vite (assumed)
